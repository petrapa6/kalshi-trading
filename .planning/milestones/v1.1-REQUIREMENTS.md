- [x] **BT-04**: User picks a `(league, season)` combination via a dropdown; selection drives which dataset is rendered
  - **Outcome: Validated** — `useMemo` re-runs `runBacktest` on `selectedKey` change; no stale state.

  - **Outcome: Validated** — grep returns 0 matches for `api/backtest/soccer` and `BankrollChart`; `pnpm build` passes clean.
